# Lucid CMS Railway

Lucid CMS starter for Railway using the Node runtime, PostgreSQL, the Pages
plugin, and persistent filesystem media storage.

## Commands

| Command | Action |
| :------ | :----- |
| `npm run dev` | Start Lucid development mode |
| `npm run build` | Build Lucid for production |
| `npm start` | Start an existing production build |
| `npm run typegen` | Generate Lucid type files |
| `npm run sync` | Sync Lucid configuration |
| `npm run migrate` | Run pending database migrations interactively |
| `npm run migrate:deploy` | Run non-destructive migrations non-interactively |
| `npm run migrate:status` | Show migration status |
